# Iris Flower Classification using Random Forest and TensorFlow

## Overview

This project applies **Random Forest Classification** and a **TensorFlow Neural Network** to classify Iris flowers into three different species based on their physical characteristics. It demonstrates a complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature scaling, model training, and model evaluation.

---

## Dataset

The dataset used in this project is the **Iris Dataset** from Kaggle.

Dataset Link:
https://www.kaggle.com/datasets/uciml/iris

## Workflow

1. Import libraries
2. Load dataset
3. Explore and preprocess data
4. Perform exploratory data analysis (EDA)
5. Encode target labels
6. Split data into training and testing sets
7. Apply feature scaling
8. Train a Random Forest Classifier
9. Train a TensorFlow Neural Network
10. Evaluate model performance
